## 🚀 Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/Abdul-razak98/velora-tech.git

# Navigate into the project
cd velora-tech/devstudio-v2/devstudio

# Install dependencies
npm install

# Run development server
npm run dev
# Runs on http://localhost:3000
```
